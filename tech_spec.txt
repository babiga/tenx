Кэтэринг Үйлчилгээний Платформ – Техникийн Тодорхойлолт

Төслийн зорилго
Энгийн хэрэглэгч болон компаниудад кэтэринг үйлчилгээ захиалах, үнэлгээ авах, тогооч сонгох, цахим гэрээ байгуулах, төлбөр хийх боломжтой веб платформ.

Технологи
- Frontend: Next.js (TypeScript)
- Backend: Express.js (TypeScript)
- Database: PostgreSQL
- Authentication: Google Login
- PWA, Push Notification, SEO

Хэрэглэгчийн төрөл
1. Хэрэглэгч (Энгийн хүн, Компани)
2. Тогооч
3. Чимэглэл хийдэг компани
4. Админ

Үндсэн функцууд

Frontend (Хэрэглэгч)
- Кэтэринг үйлчилгээ сонгох (Энгийн, VIP)
- Үнэлгээ гаргах, үнэлгээний төрөл сонгох
- Өдөр, цаг сонгох
- Нэмэлт шаардлага оруулах
- Тогоочийн танилцуулга, одоор үнэлэх
- Гүйцэтгэсэн ажлын танилцуулах хэсэг
- Цахим гэрээ хийх
- Төлбөр хийх, шилжүүлэг
- Нэхэмжлэх хүлээн авах
- Англи / Монгол хэлний дэмжлэг

VIP Кэтэринг
- VIP цэс тусдаа
- Тогооч сонгох тусдаа логик
- Өөр үнэлгээ, үнэ бодолт

Тогоочийн хэсэг
- Тогооч бүртгэх, нэвтрэх
- Профайл, портфолио удирдах
- Үйлчилгээ авсан хүмүүсийн үнэлгээ
- Жилийн хураамжтай гишүүнлэл
- Төлбөр шилжүүлсэн гүйлгээний түүх
- Ажил, эвент, сургалтын зар авах
- Админаас томилогдсон ажлын мэдэгдэл (имэйл, push)
- Job invitation хүлээн авах

Чимэглэл хийдэг компани
- Компани бүртгэх
- Үйлчилгээ, танилцуулга оруулах
- Админаар баталгаажуулах

Admin Panel
- Хэрэглэгч, тогооч, компани удирдах
- Тогооч батлах, ажлын томилгоо хийх
- Цахим гэрээ удирдах
- Төлбөр, нэхэмжлэх хянах
- VIP цэс, контент удирдах
- Гүйлгээний түүх харах

Backend
- REST API
- Authentication (Google OAuth)
- Role-based access control
- Booking, Contract, Payment, Invoice логик
- Push Notification, Email service

Database (PostgreSQL)
- users
- roles
- chefs
- companies
- bookings
- contracts
- payments
- invoices
- reviews
- memberships
- portfolios

Нэмэлт
- PWA дэмжлэг
- Push Notification
- SEO optimization
